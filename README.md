# Index.html

Hejsan Svejsan!

Detta är mitt första repo på GitHub.
Här nedanför förklarar jag hur man skapar en index.html (en textfil) och hur man skriver i filen med  Nano (ett textverktyg i terminalen på Mac och Linux). 

## Instruktioner

1. Gå in i den mappen där din du vill ha din index.html genom `cd *mappnamn*`.
2. När du är inne i din valda mapp, skapar du en index fil genom `touch index.html`.
3. skriv in `nano index.html` för att öppna html-filen. Du nu hamna i ett fönster som heter Nano, ett textverktyg i terminalen som används för att kunna skriva i html-filen.
4. Testa att skriva något. När du är klar trycker du `ctrl + o` för att spara och tyck sedan *enter*.
5. För att komma tillbaka till terminalen tryck in `ctrl + x`.



## Bonus - Git kommandon
skriv in följande i termnalen:
1. `git init` 
2. `git status` 
3. `git add index.html` 
4. `git status`
-Detta gör att Git markerar html-filen som redo för en commit, vilket betyder att ändringar kan saparas och laddas upp till exempelvis Github. 
Du kan också lägga till en kommenter om din ändring genom `git commit -m "kommentar"`
