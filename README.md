Hej!

Detta är mitt första repo på Github, här nednför förklarar jag hur man skapar en index.hmtl (vilket är en textfil) samt hur man skriver och redigerar denna fil. 

## Här nedanför är koden man kan använda i terminalen för att öppna och redigera index.html 

1. Välj den givna mappen där din index.html ligger genom `cd *mappnamn*`
2. Har du inte skapat en index fil gör du det genom `touch index.html`
3. skriv in `nano index.html` för att öppna html fil. Nano är ett skrivverktyg som man använder för att kunna skriva i html filen
4. När du är klar tryck `cmd + o` för att spara och tyck sedan enter
5. Tryck sedan `cmd + x` för att stänga ner nano
