# Index.html

Hejsan Svejsan!

Detta är mitt första repo på Github, här nedanför förklarar jag hur man skapar en index.hmtl (vilket är en textfil) samt hur man skriver i filen genom Nano (Nano finns på Mac och Linux). 

## Instruktioner

1. Välj den givna mappen där din du vill ha din index.html genom `cd *mappnamn*`.
2. När du är inne i din valda mapp, skapar du en index fil genom `touch index.html`.
3. skriv in `nano index.html` för att öppna html fil. Du kommer nu hamna i ett fönster som heter Nano, det är ett textverktyg i terminalen på Mac och Linux som man använder för att kunna skriva i en html fil.
4. Testa att skriva något. När du är klar tryck in `ctrl + o` för att spara och tyck sedan *enter*.
5. För att komma tillbaka till terminalen tryck in `ctrl + x`.



## Bonus
skriv in 
1. `git init` 
2. `git status` 
3. `git add index.html` 
4. `git status`
För att tydligt se och spara ändringar via git verktyget. Du kan även lägg till en kommenter om din ändring så det blir tydligt på branchen genom `git commit -m "kommentar"`
